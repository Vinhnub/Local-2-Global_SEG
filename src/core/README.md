# ⚙️ Core Libraries Directory

## 📌 Chức năng (What it does)
Thư mục này chứa mã nguồn (Source Code) gốc của các thư viện bên thứ 3 mà hệ thống vay mượn thuật toán.
Cụ thể, nó chứa mã nguồn của mô hình **SuperGlobal**.

> ⚠️ **LƯU Ý:** Bạn không cần và không nên chỉnh sửa bất kỳ dòng code nào trong thư mục này. Nó đóng vai trò như một Package Library.
